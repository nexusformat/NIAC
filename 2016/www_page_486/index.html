Quick search

NeXus is developed as an international standard by scientists and programmers representing major scientific facilities in Europe, Asia, Australia, and North America in order to facilitate greater cooperation in the analysis and visualization of neutron, x-ray, and muon data.

Introduction

NeXus is built on top of the scientific data format HDF5 and adds domain-specific rules for organizing data within HDF5 files in addition to a dictionary of well-defined domain-specific field names.

Reference

NeXus has defined a set of base classes that contain dictionaries of metadata required in x-ray and neutron data analysis and a set of application definitions that define what items should be present for specific analysis applications. These NXDL class definitions are defined in XML files utilizing the NeXus Data Language (NXDL)
